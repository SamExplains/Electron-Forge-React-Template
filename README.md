# Electron-Forge-React-Template
Electron forge basic template with React, Webpack and Bootstrap 5 included.

https://www.electronforge.io/

Standard Electron Forge installation with React and Webpack included following their directions. This also included Bootstrap 5. 

The purpose of this is the current Electron Forge with Webpack / React installations goes along well but there is an issue when React is introduced. To save you time this fixed that issue with the uses of this gentlements repository.

https://www.npmjs.com/package/@timfish/webpack-asset-relocator-loader

Anyways, enjoy and have fun.
